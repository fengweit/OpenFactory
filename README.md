# OpenFactory

MCP server + REST API that turns verified Shenzhen factories into callable tools for AI agents.

**What Stripe did for payments, OpenFactory does for manufacturing.**

## Tools
- `search_factories` — find verified Shenzhen factories
- `get_quote` — get a real price quote
- `place_order` — place an order with escrow
- `track_order` — track production status
