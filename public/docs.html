OpenFactory API Reference

REST + MCP. Connect your agents to GBA factories.

Quick Start
Get started in 3 steps
Install the MCP server, point it at the API, and start calling factories.
Terminal
# 1. Install the MCP server npx openfactory-mcp # 2. Point your agent to the API server export OPENFACTORY_API=https://api.openfactory.ai # 3. Start calling factories curl $OPENFACTORY_API/factories
Authentication
Bearer token auth
Register, get a token, include it in all requests.
Register & Login
# Register a new account curl -X POST /auth/register \ -H "Content-Type: application/json" \ -d '{"email":"buyer@example.com","password":"secret","role":"buyer"}' # Login to get a JWT token curl -X POST /auth/login \ -H "Content-Type: application/json" \ -d '{"email":"buyer@example.com","password":"secret"}' # Use the token in requests curl /factories \ -H "Authorization: Bearer YOUR_TOKEN"

Trust & Escrow
Trust & Escrow API
Deposit locking, dispute resolution, and escrow release endpoints.
MethodPathDescription
POST/orders/:id/escrow/lockLock 30% buyer deposit — transitions escrow to deposit_held
POST/orders/:id/disputeRaise a dispute on an order (buyer/factory/platform)
GET/orders/:id/disputeGet all disputes for an order
POST/disputes/:id/resolveResolve a dispute — refund_full, refund_partial, or rejected
POST/orders/:id/release-escrowRelease final escrow after buyer confirms receipt (requires qc_pass)
Lock deposit
curl -X POST /orders/ord-001/escrow/lock \ -H "Authorization: Bearer TOKEN" // Response { "order_id": "ord-001", "deposit_locked": true, "deposit_amount_usd": 630, "remaining_usd": 1470, "escrow_status": "deposit_held", "message": "Deposit locked. Factory can now begin production." }
Raise dispute
curl -X POST /orders/ord-001/dispute \ -H "Authorization: Bearer TOKEN" \ -H "Content-Type: application/json" \ -d '{"raised_by":"buyer","reason":"Units arrived damaged","evidence_urls":["https://photos.example.com/dmg1.jpg"]}' // Response { "id": "dsp-m3x7k9", "order_id": "ord-001", "raised_by": "buyer", "reason": "Units arrived damaged", "status": "open" }
