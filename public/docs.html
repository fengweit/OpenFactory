API Reference

Complete REST API and MCP tool documentation for OpenFactory. All endpoints return JSON. Rate limit: 100 req/min per IP. Authentication via JWT Bearer token or API key header.

Try It

Base URL is auto-detected. Paste a JWT token or API key above, then copy curl examples below.

Factories
Search, browse, verify identity, query capacity, pricing rules, and instant quotes.
GET /factories/public Public ▶

Returns a curated subset of factory data without contact information. Suitable for unauthenticated browsing.

Query Parameters
NameTypeDescription
categorystringFilter by product category
verified_onlybooleanOnly verified factories
sortstringSort field (e.g. trust_score)
min_trust_scorenumberMinimum trust score 0-100
curl
# Browse public factory listing curl "${BASE}/factories/public?category=pcb_assembly&verified_only=true"

min_trust_scorenumberMinimum trust score 0-100

{ "factories": [ { "id": "sz-001", "name": "Shenzhen Precision Electronics", "categories": ["pcb_assembly"], "moq": 500, "price_tier": "mid", "verified": true, "trust_score": 78, "rating": 4.5 } ], "count": 1 }
curl

Quotes
Request quotes from factories, respond to quotes, and broadcast RFQs.
POST /quotes JWT / API Key ▶

Request a price quote from a factory. Returns a quote_id valid for 7 days. Factory responds via WeChat notification within 2 hours.

Request Body
FieldTypeDescription
factory_idstringTarget factory ID
product_descriptionstringWhat to manufacture
quantitynumberNumber of units
buyer_idstringBuyer identifier
target_price_usdnumberTarget unit price in USD
deadline_daysnumberDays until delivery needed
Example Response
{ "quote_id": "q-abc123", "factory_id": "sz-001", "product_description": "Custom PCB assembly, 4-layer", "quantity": 5000, "unit_price_usd": 4.50, "total_price_usd": 22500, "lead_time_days": 21, "valid_until": "2026-04-02T12:00:00Z" }
curl
curl -X POST "${BASE}/quotes" \ -H "Content-Type: application/json" \ -H "Authorization: Bearer YOUR_JWT" \ -d '{ "factory_id": "sz-001", "product_description": "Custom PCB assembly, 4-layer", "quantity": 5000, "target_price_usd": 4.50 }'
