Authentication
Endpoints marked Auth Required need a Bearer token (JWT) or X-API-Key header. Register and login to get a JWT. Admin endpoints additionally require role=admin.
POST /auth/register Public Create account ▶
Request Body
JSON
{
  "email": "buyer@example.com",
  "password": "securepass123",
  "role": "buyer",        // "buyer" | "factory"
  "factory_id": "sz-001"  // required if role=factory
}
Response
201 Created
{
  "user_id": "usr-abc123",
  "token": "eyJhbGciOiJIUzI1NiIs..."
}
Status Codes
201 Created 400 Validation Error

Factory Discovery
Search and browse verified GBA factories. The public endpoint returns curated data without contact info; the authenticated endpoint returns full details including identity.
GET /factories/public Public Curated factory list (no contact info) ▶
Query Parameters
category — Product category (e.g. pcb_assembly, electronics_accessories)
verified_only — true to filter to verified only
sort — Sort field (rating, trust_score)
min_trust_score — Minimum trust score (0–100)
Response
200 OK
{
  "factories": [{
    "id": "sz-001",
    "name": "Nanshan IoT Systems",
    "location": { "city": "Shenzhen", "district": "Nanshan" },
    "categories": ["pcb_assembly"],
    "verified": true,
    "trust_score": 82,
    "rating": 4.7,
    "moq": 500,
    "lead_time_days": 21,
    "capacity_units_per_month": 100000
  }],
  "count": 16
}
Status Codes
200 OK

  "quantity": 5000,

Milestone Tracking
Factory-side production milestone reporting with photo evidence. Enforces ordering: material_received → production_started → qc_in_progress → qc_pass/fail → ready_for_shipment → shipped.
POST /orders/:id/milestones Auth Required Report production milestone ▶
Authorization
Only factory or admin roles.
Request Body
JSON
{
  "milestone": "production_started",
  "photo_urls": ["https://..."],
  "note": "Assembly line A running"
}
Valid Milestones
material_received production_started qc_in_progress qc_pass qc_fail ready_for_shipment shipped
Response
201 Created
{
  "id": 42,
  "order_id": "ord-xyz789",
  "milestone": "production_started",
  "reported_by": "usr-factory1",
  "created_at": "2025-01-20T08:00:00Z",
  "photo_upload_url": "/orders/ord-xyz789/milestones/42/photos"
}
Status Codes
201 Created 400 Invalid Milestone 403 Forbidden

Escrow
Milestone-gated payment protection. States: pending_deposit → deposit_held → production_in_progress → qc_complete → final_released.
POST /orders/:id/escrow/lock Auth Required Lock 30% deposit ▶
Locks 30% of order total as deposit. Transitions escrow from pending_deposit to deposit_held.
Response
200 OK
{
  "order_id": "ord-xyz789",
  "deposit_locked": true,
  "deposit_amount_usd": 6300,
  "remaining_usd": 14700,
  "escrow_status": "deposit_held",
  "message": "Deposit locked. Factory can now begin production."
}
Status Codes
200 OK 404 Order Not Found 409 Already Locked
POST /orders/:id/escrow/release Auth Required Milestone-based escrow release ▶
Request Body
JSON
{
  "milestone": "qc_pass",       // "production_started" | "qc_pass" | "shipped"
  "verified_by": "admin-001"
}
Status Codes
200 OK 400 Missing Milestone 404 Not Found 409 Invalid Transition
POST /orders/:id/release-escrow Public Buyer confirms receipt (final) ▶
Validates QC pass milestone before releasing. Transitions to final_released and releases Stripe payment.
Status Codes
200 OK 400 Error 409 Prerequisites Not Met
