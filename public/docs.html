Authentication

Authenticated endpoints require an API key passed via the Authorization header. Keys use the ofk_ prefix.

Header
Authorization: Bearer ofk_your_api_key_here

API keys are rate-limited (default 60 req/min). To request an API key, contact us or use the button below.

MCP Server

Connect your AI agent directly to OpenFactory via the Model Context Protocol. All V1 endpoints are exposed as MCP tools.

npx openfactory-mcp

The MCP server exposes tools like search_factories, get_instant_quote, and query_live_capacity that mirror the REST API.

Search Factories

GET /factories Auth required

Search and filter verified factories by category, MOQ, pricing tier, rating, and trust score.

Query Parameters

ParamTypeDescription
categorystringFilter by category (e.g. pcb_assembly, electronics_accessories, metal_enclosure)
max_moqnumberMaximum minimum order quantity
price_tierstringPrice tier: budget, mid, or premium
min_ratingnumberMinimum factory rating (0–5)
verified_onlybooleanOnly return verified factories (true / false)

curl

bash
# Search verified PCB factories with MOQ ≤ 500 curl "https://api.openfactory.ai/factories?category=pcb_assembly&max_moq=500&verified_only=true" \ -H "Authorization: Bearer ofk_your_api_key"

Response

200 OK
{ "factories": [ { "id": "sz-001", "name_en": "Shenzhen Rapid PCB Co.", "city": "Shenzhen", "categories": ["pcb_assembly"], "verified": true, "rating": 4.8, "moq": 100, "price_tier": "mid", "lead_time_days": 14, "identity_complete": true } ], "count": 1 }

Instant Quote

GET /factories/:id/instant-quote

Get a binding price quote in <100ms from a factory's pre-declared pricing rules. No email, no waiting. Returns pricing tiers so you can see volume discounts.

Query Parameters

ParamTypeDescription
categorystringrequired — Product category
qtynumberrequired — Order quantity (must be > 0)

curl

bash
# Get instant quote for 5,000 PCB assemblies from factory sz-001 curl "https://api.openfactory.ai/factories/sz-001/instant-quote?category=pcb_assembly&qty=5000"

Response

200 OK
{ "factory_id": "sz-001", "factory_name": "Shenzhen Rapid PCB Co.", "category": "pcb_assembly", "quantity": 5000, "unit_price_usd": 4.20, "total_price_usd": 21000, "lead_time_days": 21, "pricing_tiers": [ { "min_qty": 100, "max_qty": 999, "unit_price_usd": 5.50 }, { "min_qty": 1000, "max_qty": 4999, "unit_price_usd": 4.80 }, { "min_qty": 5000, "max_qty": null, "unit_price_usd": 4.20 } ], "binding": true, "valid_until": "2026-04-02T00:00:00Z" }

Live Capacity

GET /capacity

Query live production availability across all factories. Returns factories that have capacity for your order right now.

Query Parameters

ParamTypeDescription
categorystringrequired — Product category
qtynumberrequired — Minimum quantity needed (must be > 0)
max_daysnumberMaximum lead time in days

curl

bash
# Find factories with capacity for 1,000 PCBs shipping within 14 days curl "https://api.openfactory.ai/capacity?category=pcb_assembly&qty=1000&max_days=14"

Response

200 OK
{ "query": { "category": "pcb_assembly", "quantity": 1000, "max_days": 14 }, "count": 2, "results": [ { "factory_id": "sz-001", "factory_name": "Shenzhen Rapid PCB Co.", "available_units": 5000, "lead_time_days": 12, "unit_price_usd": 4.80 }, { "factory_id": "sz-003", "factory_name": "Nanshan IoT Systems", "available_units": 8000, "lead_time_days": 10, "unit_price_usd": 4.50 } ], "computed_in_ms": 0 }

OpenAPI Schema

The full OpenAPI 3.0 specification is available at GET /openapi.json. Use it to generate client libraries or import into Postman.

bash
curl "https://api.openfactory.ai/openapi.json" | python -m json.tool
