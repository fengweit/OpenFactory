Integration
Two MCP calls. Quote to order.
Your agent uses standard MCP tool_call protocol. Here's the full flow.
agent.ts — Step 1: Get instant quote MCP tool_call
// Agent sends MCP tool_call to OpenFactory server { "method": "tools/call", "params": { "name": "get_instant_quote", "arguments": { "factory_id": "sz-003", "category": "pcb_assembly", "quantity": 5000 } } } // OpenFactory responds in 36ms { "result": { "factory_id": "sz-003", "factory_name": "Shenzhen TechBoard Electronics", "unit_price_usd": 2.85, "total_price_usd": 14250, "lead_time_days": 18, "quote_valid_hours": 24, "binding": true, "confidence": "high" } }
agent.ts — Step 2: Place order MCP tool_call
// Agent confirms — place the order { "method": "tools/call", "params": { "name": "place_order", "arguments": { "quote_id": "q-1711234567890", "buyer_id": "lio-agent-prod" } } } // Order confirmed, escrow held { "result": { "order_id": "ord-1711234599000", "status": "confirmed", "total_price_usd": 14250, "escrow_held": true, "estimated_ship_date": "2026-04-11" } }
